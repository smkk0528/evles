selector 返回值:Array<Element(s)>

概述

根据给定的ID匹配一个元素。

描述:

查找 ID 为"myDiv"的元素。

HTML 代码:

<div id="notMe"><p>id="notMe"</p></div>
<div id="myDiv">id="myDiv"</div>

elves 代码:

$("#myDiv");

结果:

[ <div id="myDiv">id="myDiv"</div> ]

概述

根据给定的元素名匹配所有元素.

描述:

查找所有的DIV 元素。

HTML 代码:

<div>DIV1</div>
<div>DIV2</div>
<span>SPAN</span>

elves 代码:

$("div");

结果:

[ <div>DIV1</div>, <div>DIV2</div> ]
