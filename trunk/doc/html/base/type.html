type>

概述

取输入对象的类型

描述:

用法： $.type([Anything]);

可以区分以下类型：Boolean Number String Function Array Date RegExp Object Error

elves 代码:

document.write($.type(true));
document.write($.type(1.3));
document.write($.type('this is string'));
document.write($.type(function(){}));
document.write($.type([]));
document.write($.type(/a/gi));
document.write($.type({}));

结果:

Boolean
Number
String
Function
Array
RegExp
Object
